# Beryl AX HomeRacker Sleeve

This model is being rebuilt for a GL.iNet Beryl AX HomeRacker sleeve.

The current source is a blank OpenSCAD scaffold that imports the Beryl AX holster
STL from `reference/`. Subtractive cuts and the side-mounted HomeRacker sleeve
will be added in later passes.

Build from the repository root:

```sh
make build
```

Preview PNGs and exported STLs are generated under `renders/` and intentionally
ignored by Git.
