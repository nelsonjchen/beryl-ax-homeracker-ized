# Cisco AP HomeRacker Mount

This model adapts the Cisco AIR-AP-BRACKET-1 slide-hole pattern into a HomeRacker-mounted plate.

The current part slides over one HomeRacker support/bar using a U-shaped sleeve and can be secured with standard HomeRacker lock pins. The AP-facing plate includes four recessed keyhole/slider cutouts and a vent pattern to leave airflow behind the AP.

Build from the repository root:

```sh
make build
```

Preview PNGs are generated under `renders/` and intentionally ignored by Git.
