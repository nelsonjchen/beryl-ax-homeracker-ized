# Ruckus AP HomeRacker Mount

This model is being rebuilt for a Ruckus access point mount.

The current printable part is only the HomeRacker sleeve scaffold. It provides
two sleeve segments sized for a 15 mm HomeRacker bar, each with standard 4 mm
lock-pin holes. The Ruckus AP-facing geometry will be derived from the STL
reference under `reference/` in a later pass.

Build from the repository root:

```sh
make build
```

Preview PNGs and exported STLs are generated under `renders/` and intentionally
ignored by Git.
